# Hackintosh-B460M-10400-5500XT

基于MacOS 13.3和OpenCore 0.9.1

需自行修改MLB、ROM、SystemSerialNumber和SystemUUID

| 类型 | 配置 |
| --- | --- |
| CPU | intel i5-10400 |
| 主板 | 微星 B460M 迫击炮 |
| 显卡 | 蓝宝石 5500XT |
| 无线网卡 | BCM94360CD |

config针对以上配置定制，不保证其他配置可用

BIOS:
所有设置默认，然后打开D.T.M即可，路径为：Settings\Advanced，此为微星主板的黑苹果模式，打开后兼容性俱佳
